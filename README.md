# Pets

  前些天看到一个有趣的网站，提供猫咪和狗狗图片的api：
  - cat [https://thecatapi.com](https://thecatapi.com/)
  - dog [https://thedogapi.com](https://thedogapi.com/)
  
  所以用这个api来练练前端。项目是用vue + vuex开发，适合新手练习看看。
